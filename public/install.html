Install SRS Flashcards as a Home Screen App

Get the app-like experience with offline access and home screen launching.

Table of Contents

  • Android Instructions
  • iPhone/iOS Instructions
  • After Installation
  • Uninstalling
  • Learn More About PWAs (Installable Web Apps)

Android Instructions

1

Open SRS Flashcards in Chrome browser and tap the three-dot menu in the top right

2

Tap Add to Home screen

3

Select "Install app" then "Add to Home screen"

Note: If you don't see the install option, try using Chrome and ensure you've visited the site at least once before.

iPhone/iOS Instructions

1

Open SRS Flashcards in Safari and tap the share icon (box with arrow)

2

Select "Add to Home Screen"

3

Confirm by tapping "Add" in top right

Note: For best experience, iOS 16.4+ is recommended for full PWA features.

After Installation

Once installed, you can:

  • Launch like a regular app
  • Use offline during commutes or travel
  • Receive automatic updates

Uninstalling

  • Android:
    • Long-press app icon > App info > Uninstall
    • Settings > Apps > SRS Flashcards > Uninstall
    • Settings > Storage > Apps > Chrome > Clear Storage > Site Storage > Manage > SRS Flashcards > Delete Data
  • iOS: Long-press app icon > Delete App

Learn More About PWAs (Installable Web Apps)

PWAs (Progressive Web Apps) are websites that work like regular apps - they can be installed, work offline, and send notifications.
