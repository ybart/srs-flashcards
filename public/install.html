Install SRS Flashcards as a Home Screen App

Get the app-like experience with offline access and home screen launching.

⚠️ Preview Notice: This app is currently in preview. The update mechanism doesn't work properly yet - to update, you'll need to completely delete and reinstall the app. Since data restoration isn't fully implemented, this means your progress will be lost during updates.

Note: You can download your progress data for backup, but the mechanism to restore it is not yet implemented.

Android Instructions

1

Tap the three-dot menu in the top right

2

Tap Add to Home screen

3

Select "Install app" then "Add to Home screen"

Note: If you don't see the install option, try using Chrome and ensure you are on the right page.

iPhone, iOS & macOS Instructions

1

Tap the share icon (box with arrow) in the bottom (you may need to scroll up a bit to see it)

On macOS, you can also use the box with arrow button in the top right of Safari. In Chrome, there is an install button on the right side of the address bar.

2

Select "Add to Home Screen" (or "Add to Dock" on macOS)

3

Confirm by tapping "Add" in top right

Note: For best experience, iOS 16.4+ is recommended for full PWA features.

Direct Web Access

Note about web version:

This is the web version of the app. It is not as feature-rich as the installed version. It's a good way to try the app without installing it, however we recommend installing the app for the best experience.

  • Your progress will be lost when the browser clears the web site cache
  • No notifications to help you remember to review your flashcards
  • No offline access - requires internet connection
  • Some features may not work as expected

Uninstalling

  • Android:
    • Long-press app icon > App info > Uninstall
    • Settings > Apps > SRS Flashcards > Uninstall
    • Settings > Storage > Apps > Chrome > Clear Storage > Site Storage > Manage > SRS Flashcards > Delete Data
  • iOS: Long-press app icon > Delete App
